# Build 4 Starter Template

This is your starting point for Build 4: Landing Page.

## What's here

**`index.html`** — A bare HTML page with DaisyUI and Tailwind already connected via CDN. No install needed — open the file and start building.

**`styles.css`** — For the things DaisyUI doesn't handle: typeface, body-level defaults, and layout. If you need to place multiple elements side by side (cards in a row, for example), use flexbox here — same approach as Build 2. DaisyUI styles the components; CSS handles the layout between them.

## Getting started

Your full instructions are in Google Classroom. The short version: pick a subject, plan your five sections, then build them out using DaisyUI components from [daisyui.com/components](https://daisyui.com/components).

To apply a theme, add `data-theme="[name]"` to the `<html>` tag. Browse themes at [daisyui.com/docs/themes](https://daisyui.com/docs/themes).

Not sure what an attribute does, and it's not in the DaisyUI docs? Search for `tailwind [attribute]`.
